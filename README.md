#Pig game

It is a two-player web-game that works following these rules: 
- First to reach a score of 100 wins
- The dice switches to opponent automatically if it lands on 1
- Once you hold, you switch to the other player

This was the 3rd project in the "Complete javascript Course 2025" by Jonas Schmedtmann and can be accessed [here](https://dice-game-david.netlify.app/)

This project focused on DOM manipluation
